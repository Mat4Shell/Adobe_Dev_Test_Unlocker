# Installation Guide

## Prerequisites

- C++17 compiler
- Python 3.8+
- CMake 3.15+

## Build Steps

### Linux/Mac

```bash
mkdir build
cd build
cmake ..
make
```

### Windows

```bash
mkdir build
cd build
cmake ..
cmake --build .
```

## Python Setup

```bash
pip install -r requirements.txt
python setup.py install
```
